# LED Banner Tool 产品需求文档

## 1. 产品概述
LED Banner Tool 是一款专业的LED滚动字幕制作工具，可将设备瞬间变为炫酷的LED显示屏招牌。
- 主要用于演唱会应援、活动引导、生日祝福、派对氛围、店铺招牌等场景
- 支持自定义文字样式、滚动动画、边框特效、背景效果，适配PC、平板、手机全平台

## 2. 核心功能

### 2.1 功能模块
1. **预览展示区**：实时LED效果预览、全屏播放模式
2. **文字设置面板**：文字内容输入、字体样式、大小、颜色、特效
3. **背景设置面板**：背景颜色、LED点阵、背景图片/GIF、亮度调节
4. **滚动控制面板**：滚动方向、速度、闪烁效果
5. **边框效果面板**：多种霓虹边框、装饰边框样式
6. **模板/保存**：预设模板选择、作品本地保存

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主编辑页 | 预览区 | 实时渲染LED效果，支持点击全屏播放 |
| 主编辑页 | 标签切换栏 | Text/Background/Control/Effect四个功能标签切换 |
| 主编辑页 | 文字设置面板 | 字体选择、字号滑块、颜色拾取、粗体/斜体/对齐按钮、描边、阴影开关 |
| 主编辑页 | 背景设置面板 | 背景颜色选择、LED点阵效果开关、亮度调节、背景图选择 |
| 主编辑页 | 控制面板 | 滚动方向(左/右/上/下/静止)、速度滑块、闪烁效果开关 |
| 主编辑页 | 效果面板 | 多种霓虹边框样式选择、特殊光效选择 |
| 全屏播放页 | 播放控制 | 点击暂停/继续、双击退出全屏 |

## 3. 核心流程
用户打开页面 → 输入文字内容 → 调整文字样式(字体/颜色/大小) → 选择背景和边框效果 → 设置滚动方向和速度 → 点击全屏播放展示LED效果

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深黑底色(#0a0a0f)搭配霓虹渐变(紫#b026ff、青#00f5ff、粉#ff2d92)，营造真实LED氛围
- **按钮风格**：圆角胶囊按钮，选中态霓虹发光效果，按压反馈
- **字体**：标题使用Orbitron(未来科技感)，正文使用Inter，文字预览支持多种字体
- **布局**：上预览下编辑的经典移动端布局，PC端左右分栏布局
- **视觉风格**：赛博朋克霓虹风，点阵纹理背景，发光边框，深色主题

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|-------|
| 主编辑页 | 顶部导航栏 | LED Logo、应用标题、全屏按钮 |
| 主编辑页 | LED预览区 | 黑色圆角矩形、点阵纹理、文字滚动动画、霓虹边框 |
| 主编辑页 | 标签栏 | 四个胶囊标签(Text/Background/Control/Effect)，选中态霓虹发光 |
| 主编辑页 | 编辑面板 | 深色卡片式设计、滑块控件、颜色选择器、开关按钮、选项网格 |
| 全屏播放页 | 播放区域 | 纯黑背景、LED文字居中/滚动、无UI元素、点击暂停图标 |

### 4.3 响应式设计
- 采用Desktop-first设计原则，桌面端左右分栏(预览区左、编辑区右)
- 平板端自适应布局，保持操作舒适度
- 手机端上下布局：上方预览区占40%，下方编辑区可滚动，触控优化按钮尺寸≥44px
- 全屏播放模式在所有设备上自动旋转适配横屏/竖屏
- 支持触摸滑动调节滑块、点击反馈动画
